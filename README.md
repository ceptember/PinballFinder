# PinballFinder

## About the Code 

This app was written in React and bootstrapped with [Create React App](https://github.com/facebook/create-react-app). 

To view the code, please open the file `./src/App.js` 


## Creating a local copy 

### Setup 
Requirements: [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

Clone this project into your local environment. Navigate to the project directory in the command line, and run `npm install`, which will install the dependencies listed in the `package.json` file.


### Deploying to local host 

From the project directory, run `npm start`, which runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.
